# AI-Chatbot
#### The Chatbot uses ollama3.2 model to answer your queries.
#### When you upload an xlsx file, a vector DB will be created which will locally persist on your system, and until you upload another file it will take context from the pre-existing vector DB.
#### I have also implemented chatbot history.
#### You need to install ollama application on your computer first and then run "ollama run llama3.2"  and 'pull nomic-embed-text' on cmd to download the actual model.
#### Based on the xlsx file, the vector DB will take some time to be created
